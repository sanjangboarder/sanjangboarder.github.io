Hello, this is SanjangBorder.

​

Today, I introduce a new product scheduled for release tomorrow, October 21st, 2025: the **DelliRig ViviTune (BB Custom)** from **Tulbo Fishing**. Following the first and second generations of the popular DelliRig line, this version offers a simplified design.

​

While the previous DelliRig made it easy to adjust both the sinker height and branch line length, it was relatively difficult to untangle when knotted. I concluded that adjusting only the branch line length would be sufficient for most conditions, a view shared by other anglers.

​

On YouTube, Pro Angler Ko Yoon-jung shared a video demonstrating a customized DelliRig setup. Tulbo Fishing recognized the utility of this modification and collaborated with him to release the DelliRig ViviTune.

​

This new release is sold in sets of three for 4,500 KRW. Let's analyze its features in detail.

Unlike the first and second generations of the DelliRig, the ViviTune is available in three specific configurations:

1.  **Sinker height 0 cm / Max branch line 60 cm**
2.  **Sinker height 15 cm / Max branch line 60 cm**
3.  **Sinker height 20 cm / Max branch line 60 cm**

The key structural feature of the DelliRig BB Custom is shown below. A carbon line is tied to the main connection block using a stopper knot configuration, allowing for adjustment of the branch line length.

​

Sliding the knot changes the branch line length, and tightening it fixes the position. This mechanism is identical to the standard DelliRig.

​

The DelliRig BB Custom focuses on simplicity, retaining the main adjustment mechanism of the original DelliRig in a streamlined format. Because anglers adjust the branch line length more frequently than the sinker height, this design represents a practical balance.

​

Configured with zero sinker height, it serves as an effective rig for webfoot octopus, while adding sinker height optimizes it for cuttlefish. I have four octopus and cuttlefish trips remaining this season and plan to evaluate these rigs in detail.
