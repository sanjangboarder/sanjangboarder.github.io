Hello, this is SanjangBorder.

​

As winter approaches, it is time to set up heated blankets (*electric blankets / 전기장판*). Reaching behind the bed frame or the sofa to access wall outlets is inconvenient, so I purchased a dedicated extension power strip.

​

My requirements were specific: a flat plug to minimize protrusion from the wall when plugged in behind furniture, and a 1.5m to 2m extension cable. Since it is intended for a heated blanket, it required a high maximum wattage capacity. I selected the product below.

​

This is the **Philips High-Power Travel Multi-Tap 1-Outlet**. As shown on the packaging, it features integrated USB charging ports.

After approximately one month of use, the unit has functioned reliably. Using it to run the heated blanket while charging mobile devices via the USB ports has been convenient. For users needing a heavy-duty power extension cable that can fit in tight spaces, this is a practical choice.
