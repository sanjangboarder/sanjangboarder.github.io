안녕하세요 산장보더입니다.

이번에 PC 조립하면서 그 후기들을 하나씩 정리하고 있는데요.. 이번 포스팅에서는 기본적인 테스트를 마친 조립 PC를 안정화 하는 내용을 포스팅 할려고 합니다.

온도를 잡기위해서 가장 좋은 솔루션은 수냉3열 쿨러를 사용하라는게 공식처럼 되어 있지만 저는 k 버전을 오버클럭킹을 하기 위해서 구입한게 아니라 그냥 현재 non-k 13세대 cpu가 없어서 구입한거라... 오버클럭은 전혀 고려대상이 아니였습니다. 그래서 수냉쿨러를 쓸 생각이 1도 없었던 거구요..

그런데 13600k 가 기본적으로 오버클럭킹 없이도 발열이 심하니 몬가 대책이 필요했습니다.

TDP 수치만 보고 처음에 구입한 팔라딘 400 쿨러는 그냥 바로 장터행이 되었고 공랭 대장급 가성비 쿨러인 AG620으로 공랭 쿨링을 구성하고 언더볼팅을 통해 낮추면 되지 않을까 하는데서 이 작업이 시작되었습니다.

변경 셋팅값 정리

위 실험내용을 보기 좋게 표로 정리해 봤습니다.

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
|  | **CPU MAX 온도**  **(C)** | **CPU 전력량**  **(W)** | **P-Core 평균 클럭**  **(MHz)** | **CineBench 점수**  **(pt)** |
| **전력제한 없음 + mode 12 (auto) 비교군** | 96 | 192.28 | 5088 | 24098 |
| 전력제한 PL1: 288W + mode 12 (auto) | 97 | 192.96 | 5088 | 24013 |
| 전력제한 PL1: 288W + mode 9 | 94 | 193.02 | 5088 | 23899 |
| 전력제한 PL1: 288W + mode 8 | 92 | 186.51 | 5088 | 23541 |
| 전력제한 PL1: 288W + mode 7 | 87 | 173.37 | 5088 | **22230** |
| 전력제한 PL1: 181W + mode 12 (auto) | 85 | 161.67 | 4788 | 23467 |
| 전력제한 PL1: 181W + mode 8 | **81** | 159.33 | 4888 | **23296** |

일단 기본세팅과 쿨러설정을 통한 전력제한 288W는 결과값이 비슷합니다. 타워 에어쿨러로 바꾸는게 큰 의미가 없는상황인거고.. 이건 CPU 전력량이 기본셋팅에서 192를 찍기 때문인듯 합니다. 그래서 의미있는 전력제한을 하기 위해서는 박스쿨러를 선택하여 전력은 181W로 제한해하 함을 알 수 있습니다.

전력제한은 CPU Lite Load 메뉴로도 가능은 했느데요 이건 CPU 성능에 또 영향이 큰것으로 보이기 때문에 적절한 값을 찾는게 중요해 보였습니다.

표를 유심히 보면 재밌는 패턴이 볼수 있었는데요.. CPU 전력량이 낮아질 수록 CPU MAX온도가 낮아지는건 확실합니다. 그리고 전력량이 높은면 보통 cinebech 점수도 그에 비례하지만 CPU Lite Load 메뉴 설정에 따라 그 패턴이 깨지는 경우도 있었습니다.

제가 내린 결론은 전력제한 PL1: 181W + mode 8 이 제일 효율적일것 같다였습니다. 각 모드들의 비교는 시간이 많이 소요되오 Cinebench를 5분 내외를 돌리면서 확인한 사항이라 실제 사용시에서 다른 결과를 보일 수도 있기 때문에 오랜시간 테스트에도 비슷한 결과값을 보이는지 확인이 필요했습니다.
